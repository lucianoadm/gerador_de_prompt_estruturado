# gerador_de_prompt_estruturado
Gerador criado para facilitar e direciona melhor o uso da IA no dia a dia.
🚀 R-C-O-P-F: Gerador de Prompts Estruturados
Este projeto apresenta um Prompt de Autopreenchimento projetado para transformar qualquer IA (ChatGPT, Claude, Gemini) em uma ferramenta de engenharia de prompt. O objetivo é eliminar respostas genéricas através da estrutura R-C-O-P-F (Role, Context, Objective, Parameters, Format).
📋 O que é o Framework R-C-O-P-F?
Para extrair o máximo de uma IA, não basta dar uma ordem; é preciso fornecer estrutura. Este framework organiza a demanda em 5 pilares:
Role (Papel): Define a "persona" e o nível de expertise da IA.
Context (Contexto): Explica o cenário, o público e o histórico.
Objective (Objetivo): Declara exatamente qual é a entrega esperada.
Parameters (Parâmetros): Define regras, tom de voz, limites e o que evitar.
Format (Formato): Estipula como o conteúdo deve ser visualmente entregue (Tabela, Markdown, Código, etc).
🛠️ Como Utilizar
Basta copiar o "Prompt Mestre" abaixo e colá-lo em uma nova conversa com sua IA de preferência.
O Prompt Mestre:
"Aja como um especialista em Engenharia de Prompts. Seu objetivo é me ajudar a construir o prompt perfeito usando o framework R-C-O-P-F. Siga estas etapas:
Pergunte-me, um por um, o conteúdo para os campos: Role, Context, Objective, Parameters e Format.
Após eu responder a cada um, analise minhas respostas e sugira melhorias se algo estiver vago.
Ao final, gere um bloco de texto final estruturado e otimizado, pronto para ser copiado e colado em uma nova sessão de IA."
🎯 Aplicações
Este método é ideal para:
Marketing: Criação de copies, roteiros e estratégias de conteúdo.
Desenvolvimento: Escrita de documentação, refatoração de código ou arquitetura de sistemas.
Educação: Planos de estudo personalizados e explicações de conceitos complexos.
Negócios: Redação de e-mails corporativos, relatórios e análise de dados.
📈 Resultados Esperados
Ao utilizar este script de autopreenchimento, você notará:
Redução de Alucinações: A IA inventa menos fatos quando tem um contexto claro.
Menos Refação: Chega-se ao resultado desejado na primeira tentativa (Zero-shot) ou com ajustes mínimos.
Consistência: Respostas com o tom de voz e formatação idênticos em diferentes sessões.
Profundidade: Saídas muito mais ricas do que comandos simples como "escreva um texto sobre X".
